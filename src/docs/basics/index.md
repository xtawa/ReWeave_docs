# 基础指南

本章节包含了使用 ReWeave 进行日常内容创作和站点管理所需的基础知识。

## 目录

- **[项目结构](/basics/project-structure)**
  详细解析 ReWeave 项目的目录结构，帮助您快速定位配置文件、内容文件和主题代码。

- **[编写内容](/basics/writing-content)**
  学习如何使用 Markdown 编写文章和文档，包括 Frontmatter 的设置、代码高亮、数学公式以及 Mermaid 图表的使用方法。

- **[配置指南](/basics/configuration)**
  全面的配置说明，涵盖全局站点配置（标题、描述、语言）以及各个主题的专属配置选项。

## 推荐阅读顺序

建议新用户首先阅读 **项目结构** 以建立整体认知，然后通过 **编写内容** 开始创作您的第一篇文章。当您需要对站点进行个性化定制时，再查阅 **配置指南**。
