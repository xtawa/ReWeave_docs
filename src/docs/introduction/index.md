# 欢迎使用 ReWeave

**ReWeave** 是一个现代化、高性能的静态站点生成器（SSG），专为构建技术博客、项目文档和产品落地页而设计。它结合了 React/Preact 的组件化开发体验和 Markdown 的便捷写作体验，旨在提供极速的构建速度和卓越的用户体验。

## 核心设计理念

### 🚀 极致性能
ReWeave 的构建核心采用了多线程 Worker Pool 架构。通过并行处理 Markdown 解析、HTML 生成和资源优化，即使是包含数千篇包含复杂数学公式和代码块的文章的大型站点，也能在数秒内完成构建。实测 1000 篇复杂文章构建仅需 10 秒左右。

### 📝 Markdown 优先
我们深知内容创作的重要性。ReWeave 原生支持 GFM（GitHub Flavored Markdown），并内置了对 LaTeX 数学公式（KaTeX）、代码高亮（Highlight.js）、Mermaid 图表的支持。无需繁琐配置，开箱即用。

### 🎨 灵活的主题系统
ReWeave 内置了多款精心设计的主题，涵盖了从极简博客到复杂文档站点的各种需求。
- **Weave**: 默认博客主题，简洁优雅。
- **GitBook**: 专业的文档主题，支持多级目录。
- **Landing**: 现代化的产品落地页。
- **Butterfly**: 视觉效果华丽的个人博客。

此外，基于 TypeScript 和 Preact 的组件化架构使得开发自定义主题变得异常简单。

### 🛠 开发者友好
- **TypeScript**: 全程类型安全，减少运行时错误，提升开发体验。
- **Tailwind CSS**: 内置原子化 CSS 引擎，快速定制样式，易于维护。
- **热重载 (HMR)**: 毫秒级的开发响应速度，所见即所得。

## 适用场景

- **个人博客**: 使用 `weave` 或 `butterfly` 主题，快速搭建展示个人技术和生活的空间。
- **技术文档**: 使用 `gitbook` 主题，构建结构清晰、易于检索的项目文档库。
- **开源项目主页**: 使用 `landing` 主题，配合 `gitbook` 主题的文档部分，打造专业的开源项目门户。
