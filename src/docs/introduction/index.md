# 欢迎使用 ReWeave

**ReWeave** 是一个现代化、高性能的静态站点生成器（SSG），专为构建博客、文档和落地页而设计。它结合了 React/Preact 的组件化开发体验和 Markdown 的便捷写作体验，旨在提供极速的构建速度和优秀的用户体验。

## 为什么选择 ReWeave？

- 🚀 **极速构建**：利用多线程 Worker 和高效的缓存机制，即使是包含数千篇文章的大型站点也能在数秒内完成构建。
- 📝 **Markdown 优先**：原生支持 GFM（GitHub Flavored Markdown）、数学公式（KaTeX）、代码高亮等丰富特性。
- 🎨 **灵活的主题系统**：内置多款精美主题（Weave, GitBook, Butterfly, Landing），并支持完全自定义的主题开发。
- 🛠 **开发者友好**：基于 TypeScript 和 Tailwind CSS，提供类型安全和原子化 CSS 开发体验。
- 📱 **响应式设计**：所有内置主题均完美适配移动端和桌面端。

## 适用场景

- **个人博客**：使用 `weave` 或 `butterfly` 主题，快速搭建个性化博客。
- **技术文档**：使用 `gitbook` 主题，构建结构清晰、易于导航的项目文档。
- **产品落地页**：使用 `landing` 主题，展示产品特性和最新动态。
